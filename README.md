# Neovim Config

This repository contains my personal Neovim configuration. It's designed to be lightweight, efficient, and highly customizable to enhance the coding experience.

## Features

- **Efficient Keybindings**: Optimized key mappings for faster navigation and editing.
- **LSP Integration**: Language Server Protocol (LSP) integration for intelligent code completion, diagnostics, and refactoring.
- **Syntax Highlighting**: Enhanced syntax highlighting for various programming languages.
- **Git Integration**: Seamless integration with Git for version control operations.
